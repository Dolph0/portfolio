Proyectos

Web Gestiona Mogán

El proyecto consiste en una web multifuncional para Gestiona Mogan, una empresa pública que administra servicios municipales.

Coordinar múltiples entidades, incluido el organismo público, empresas externas y nuestro equipo, fue crucial para garantizar una ejecución fluida.

Las variadas características del producto, adaptadas a diferentes targets, junto con la incorporación de funcionalidades heredadas de productos anteriores del cliente aumentaron la complejidad del proyecto.

El desarrollo de este proyecto se realizó utilizando Next.js, Typescript y Tailwind CSS.

Parking Mogán

La funcionalidad principal de este proyecto es la gestión de reservas online así como el pago de las mismas para un parking público.

Coordinando múltiples aspectos, desde el diseño de los formularios hasta la implementación de la pasarela de pagos, se garantiza una ejecución fluida y una experiencia intuitiva para los usuarios.

Utilizando tecnologías de como Next.js, Tailwind CSS y Typescript, se ha creado una plataforma robusta y segura para la gestión de reservas online.
